# VoiceClaw Landing Page

Concept landing page for VoiceClaw, an AI receptionist for residential painters.

## Stack

Single self-contained HTML file. TailwindCSS (via CDN), Inter (Google Fonts). No build step, no dependencies.

## View it

- **Locally**: open `index.html` in any modern browser
- **Live**: https://ianpilon.github.io/voiceclaw-painter-landing-concept/
